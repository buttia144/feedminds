FeedingMinds Metadata Testing Guide

This guide will help you test the metadata implementation for social media sharing. We've added Open Graph and Twitter Card meta tags to improve how links appear when shared on social platforms.

Note: You'll need to deploy your website to a public URL before testing with these tools, as they need to access your site online.

Testing Tools

1. Facebook Sharing Debugger

Use this tool to test how your links will appear when shared on Facebook and other platforms that use Open Graph tags.

https://developers.facebook.com/tools/debug/

Instructions:

  • Enter your website URL
  • Click "Debug"
  • If needed, click "Scrape Again" to refresh Facebook's cache

2. Twitter Card Validator

Test how your links will appear when shared on Twitter.

https://cards-dev.twitter.com/validator

Instructions:

  • Enter your website URL
  • Click "Preview card"

3. OpenGraph.xyz

A comprehensive tool to preview how your site will appear across multiple platforms.

https://www.opengraph.xyz/

Instructions:

  • Enter your website URL
  • View previews for different platforms

4. LinkedIn Post Inspector

Test how your links will appear when shared on LinkedIn.

https://www.linkedin.com/post-inspector/

Instructions:

  • Enter your website URL
  • Click "Inspect"

5. Meta Tags Generator

If you need to make adjustments, this tool can help generate the correct meta tags.

https://metatags.io/

What to Check

  • Title appears correctly
  • Description is complete and not truncated
  • Logo image appears properly and isn't cropped oddly
  • URL is correct

Implemented Meta Tags

We've implemented the following meta tags across the main pages:

  • Basic meta tags (title, description)
  • Favicon and Apple Touch icons
  • Open Graph tags for Facebook/LinkedIn
  • Twitter Card tags

If you need to make changes to the metadata, edit the <head> section of each HTML file.
